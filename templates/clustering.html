Clustering Analysis with K-Means

Beyond classification, I explored the natural groupings of Steam games using K-Means clustering. This unsupervised learning approach was applied to numerical features (e.g., price, achievements) to identify patterns among games.

To determine the optimal number of clusters, I used the elbow method, which suggested k=4—aligning with the four genres in the dataset. Below, you’ll find the elbow plot and a scatter plot visualizing the clusters in a 2D space (using PCA for dimensionality reduction).
